Account Ledger

With help of this app we can have the simplest form of reporting for every chart of account we have in our ERP system.



Account Ledger Menu

After Installation of the this app, a new menu will appear in the Menu bar of the Invoicing App with the name of Account Ledger.

Account Ledger

Upon clicking Account ledger, a next window will pop-up in which we can select needed account ledger name and period from which we want to see the report.
